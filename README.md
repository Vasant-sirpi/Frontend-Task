# My Company - Multi-Page Static Website

## 📌 Project Overview

This project is a modern multi-page static website built using **HTML**, **CSS**, and **Tailwind CSS**. It consists of three pages with a consistent user interface and responsive design.

> **Developer Motto:**
> *"AI writes the first draft. We make it production-ready."* 🤖

---

## 📂 Project Structure

```
MyCompany/
│
├── index.html
├── about.html
├── contact.html
├── styles.css
├── README.md
└── image/
    └── images.png
```

---

## 🚀 Features

* Multi-page website
* Responsive layout using Tailwind CSS
* Common Header, Navigation Bar, and Footer
* Home page with banner image
* About page with company information and mission
* Contact page with contact form
* Google Fonts (Poppins)
* Hover effects on navigation links and buttons
* Smooth transitions and modern UI
* External CSS file for custom styling

---

## 🛠 Technologies Used

* HTML5

* Tailwind CSS
* Google Fonts (Poppins)

## 🎯 Objective

The goal of this project is to demonstrate the creation of a clean, responsive, and user-friendly multi-page website using HTML, CSS, and Tailwind CSS while maintaining a consistent design across all pages.
